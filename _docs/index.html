Tdoc

开始

TDoc

简介

前端工程文档生成工具

安装

npm install tdoc [-g]

使用方式

命令

cd /path/to/project/
tdoc build

详细请查看命令使用说明。

脚本

var tdoc = require("tdoc");

tdoc.build('/path/to/project', options);

Gulp

var tdoc = require("tdoc");

gulp.task('tdoc', function() {
    return gulp.src('./')
        .pipe(tdoc({
            // 配置
        }));
});

Grunt

grunt.initConfig({
    tdoc: {
        // 配置
    }
});

grunt.loadNpmTasks('tdoc');

配置及配置文件请查看 配置说明;
