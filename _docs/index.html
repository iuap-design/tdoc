Tdoc

开始

简介

前端工程文档生成工具

安装

npm install tdoc-cli [-g]

使用方式

  • cd /path/to/project/ 进入项目目录
  • tdoc init 初始化tdoc配置文件，编写配置文件tdoc.config (配置及配置文件请查看 配置说明)
  • tdoc build 构建文档
  • tdoc server 启动服务，访问文档页面效果

详细请查看命令使用方式说明。

使用其他方式

  1. 使用脚本：

    var tdoc = require("tdoc");

tdoc.build('/path/to/project', options);

  2. 使用Gulp：

    var tdoc = require("tdoc");

gulp.task('tdoc', function() {
    return gulp.src('./')
        .pipe(tdoc({
            // 配置
        }));
});
